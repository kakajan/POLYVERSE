# POLYVERSE — The 3-Year C# and Unity Game Developer Course

A structured, project-driven three-year journey to becoming a professional Unity game developer — built around C#, Unity, industry workflows, and real-world projects.

---

## Course Overview

POLYVERSE is a three-year, hands-on curriculum designed to take learners from beginner programmers to professional Unity game developers. You will learn C# fundamentals, Unity engine systems, gameplay programming, optimization, online multiplayer, and launch-ready game design.

This repository contains learning materials, sample projects, and a clear roadmap to guide motivated learners through the entire process.

---

## Who is this for?

- Aspiring game developers who want a focused path to learn Unity and C#.
- Hobbyist programmers wanting to transition into game development.
- Students or career changers seeking a step-by-step structured learning path and portfolio guidance.

## Prerequisites

- Basic familiarity with computers and file systems.
- A willingness to learn programming and game design.
- No prior Unity or C# experience required — we start from fundamentals.

## Learning Outcomes

By the end of Polyverse, you will be able to:

- Build playable games in Unity using C#.
- Implement gameplay systems (movement, combat, UI, AI).
- Create polished game-levels and art integration pipelines.
- Use Git for version control and follow a collaborative workflow.
- Optimize games for performance and mobile platforms.
- Design, implement, and release a capstone game with networking and analytics.

---

## Roadmap (Three Years)

Year 1 — Foundations

- Semester 1: C# Fundamentals & Unity Basics
- Semester 2: Gameplay Basics & Tools

Year 2 — Intermediate

- Semester 3: Gameplay Systems & Level Design
- Semester 4: Tools, UI, Analytics, Polish

Year 3 — Advanced & Professional

- Semester 5: Multiplayer & Advanced Systems
- Semester 6: Capstone, Portfolio & Launch
